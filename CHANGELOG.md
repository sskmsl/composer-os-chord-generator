# Changelog

Composer OS Chord Generator の開発記録。[Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) の形式に緩くならっている。

## v0.1 の後(未リリース分、時系列)

### 曲の構成パネルの機能拡張
- 曲全体のメモ欄を追加(構成の意図・アレンジのアイデアを記録)
- セクションの複製機能(直後の位置に挿入)
- セクションの直接削除機能

### 表示・操作の不具合修正
- PC版Chromeでマウスホイール/矢印キーによるスクロールが効かない不具合を修正
  (`html`/`body`両方への`overflow-x`指定が原因。`html`のみに一本化)
- ヘッダーメニューが画面幅からはみ出し、ページ全体が横に動いてしまう不具合を修正
  (ヘッダー内だけを`overflow-x-auto`でスクロール可能にし、ページ本体は
  `overflow-x: hidden`で固定)
- スマホでのピンチズーム無効化・iOSのバウンススクロール抑制・
  ステータスバー(時計等)との重なりを解消(セーフエリア対応)

### コード進行生成
- 「次のセクションのおすすめ」機能を追加。生成結果の下に王道の曲構成
  (Intro→A→B→サビ→A→B→サビ→C→落ちサビ→Outro)に沿った次の
  セクション候補を表示し、ワンクリックで生成できるように

### MIDI書き出し
- 各トラック(Chords/Bass)にセクション名のメモ書き(Text meta event)を埋め込み、
  Logic Pro等でトラック単体を見ても今どのパートか分かるように
- MIDI書き出し時に保存先を選べる保存ダイアログに変更(Electron版)
  (以前は`~/Downloads`へ自動保存)
- コードのベース音を専用トラックに分離して書き出し(スラッシュコード対応)

### デザイン
- アプリアイコンを「三日月+星」の装飾的なデザインから、太いセリフ体
  (Bodoni)の「C」一文字によるファッションブランド風のミニマルな
  モノグラムに変更

### 配信・デプロイ
- PWA化(manifest・Service Worker・アイコンを追加し、スマホのブラウザから
  ホーム画面にインストール可能に)
- GitHub Pages への自動デプロイワークフローを追加
  (`main`へのpushでGitHub Actionsが自動ビルド&デプロイ)
- macOSデスクトップアプリ化(Electronでラップし`.app`としてパッケージング)

### 安定性(Codexによるレビュー修正を含む)
- 保存済み進行の二重保存を防止(冪等化)
- フォルダ切り替え時に未保存のメモ入力が消える不具合を修正
- 音声再生の起動に失敗した際、再生状態が固まらないよう修正

## v0.1(初回リリース)

- ルールベースのコード進行生成エンジン(7スタイル × 6セクション × 8ムード、
  AI不使用)
- ディグリー解析・移調(実用24キー、フラット/シャープ表記)
- 5軸スコアリング(Mylène/Boutonnat/Melancholy/Darkness/Cinematic)と
  説明文の自動生成
- Web Audio APIによるコード進行の試聴機能(内蔵シンセ、外部音源不要)
- セクション追加(Verse 1/2/3, Final Chorus 等)・フォルダ(曲)管理・
  鍵盤ポジション表示
- SMF (MIDI) Type 1 形式での曲構成書き出し
  (Track 0=コンダクター、1コード=1小節、セクションマーカー付き)
- IndexedDBによるローカル永続化
